Básico

<!DOCTYPE html>
Esta tag serve para indicar para o navegador o tipo de arquivo que ele está prestes a abrir, sendo algo que não aparecerá de forma natural para o usuário final.
<html lang="pt-br"> e </html>
A tag html é usada para indicar o tipo de estrutura que será construída ao decorrer do site, também sendo um conteúdo que a princípio não será visto pelo usuário. Nesta área também será definida a língua referente ao texto da página, através do "lang" que no nosso caso recebe "pt-br", pois o texto será em português brasileiro.
<head> e </head>
É uma tag onde o desenvolvedor escreverá meta tags, que serão explicadas adiante, mas serve como uma parte de direção, como se fosse um mapa, onde estarão recursos para integração com outros links e que ajudarão o SEO a entender melhor seu site.
<meta>
As meta tags são usadas para descrever os elementos de uma página web, elas são lidas pelo seu navegador e interpretadas, também sendo tags as quais o usuário não vê.
<meta charset="UTF-8">
É utilizada para informar os tipos de caracteres que serão usados no código, serve para o navegador otimizar a leitura de forma mais ágil.
<meta http-equiv="X-UA-Compatible" content="IE=edge">
É uma meta tag específica para o navegador Internet Explorer que por não suportar o doctype para HTML5, tenta fazer com que seu site seja renderizado de forma a perder algumas funcionalidades e com essa tag você força o navegador a interpretar o documento da forma correta.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
A primeira parte do "content", a parte antes da vírgula, significa que a largura do site irá se adaptar de acordo com a largura do dispositivo que está realizando o acesso, já a segunda parte representa a quantidade de zoom inicial ao abrir o site pela primeira vez, que por padrão vem sem zoom.
<meta name="author" content="Nome do autor do site">
É definida para deixar registrado o nome do autor do site, sendo de uso facultativo e útil para o google reconhecer os sites feitos pelo desenvolvedor. Sendo a parte do "content" o local onde a pessoa ou a empresa coloca o seu nome.
<meta name="description" content="Descrição do site">
Está meta tag é onde o desenvolvedor coloca a descrição do site, referindo-se ao conteúdo presente no site, sendo a parte "content" onde se escreve esta descrição.
<meta name="keywords" content="Palavras chaves para encontrar o site">
É utilizada para escrever as palavras chave para melhorar as buscas pelos usuários, palavras que contenham características do site, objetivos, coisas do tipo, seguindo o mesmo padrão das tags anteriores de colocar as palavras-chave dentro do "content".
<body> e </body>
Esta é a região onde o seu site será feito de fato, região esta que será visível para o usuário final, zona que abrigará interações e etc...
<link>
Esta tag é utilizada para diversos fins, mas os principais deles são, fazer a conexão com seu favicon, que é aquele pequeno ícone que fica ao lado do nome do site no separador do navegador, e as folhas de estilo, também conhecidas como CSS.
<title> e </title>
É a tag utilizada para definir o título do seu site, aquele que aparecerá na parte de cima do navegador ao lado do favicon.

Divs

<header> e </header>
A tag acima funciona como o cabeçalho do site, local onde normalmente ficam localizados itens como a logo da marca, barra de pesquisa, menu de navegação, tema do site entre outras coisas.
<nav> e </nav>
Já esta é a tag de navegação do site a qual funciona como um gps para que o usuário consiga navegar pelo site de forma mais intuitiva. Pode se encontrar tanto dentro quanto fora da tag header.
<main> e </main>
Esta tag é onde a parte principal do seu site estará dentro, sendo aqui o local onde as outras tags se encontrarão dentro, com exceção apenas das tag header e nav.
<section> e </section>
É uma seção do seu site, normalmente uma área onde será colocado o texto ou outras "caixas" que farão o enquadramento do texto.
<article> e </article>
O article é outra tag que que cria uma espécie de caixa, porém, normalmente se encontra dentro de uma section ou vice-verse, essas tags fazem papel de "pai" e "filho", onde uma normalmente é usada para ajustar a outra.
<aside> e </aside>
Já esta tag tem como função falar de algo que não está tão relacionado com o conteúdo principal, funcionando como uma outra caixa, avulsa em relação a tag article.
<div> e </div>
Esta tag não possui significado semântico, mas para a estrutura do site ela funciona como se fosse uma caixa, guardando os itens do interior dela e realizando quebras de linha como se fosse realmente um bloco.
<span> e </span>
Esta tag possui o funcionamento similar ao de uma div, porém, não possui a função de criar uma "caixa" ao decorrer do texto, ou seja essa tag é do tipo inline box. Funciona para marcar itens específicos.
<footer> e </footer>
Este é o rodapé do site, a parte onde normalmente se encontram informações de contato, horário de funcionamento dos canais de comunicação, direitos autorais e entre outras informações.

Parágrafos

<h1> e </h1> até <h6> e </h6>
Estas são as tags de título sendo a h1 a com maior "importância" dentro do site, para o navegador, pois o tamanho e a fonte são facilmente alteradas com CSS. Variam do h1 até o h6, sendo mais usadas do h1 até o h3 normalmente.
<p> e </p>
Tag para inserir um parágrafo no seu texto.
<strong> e </strong>
Esta tag é responsável por deixar seu texto em negrito, de forma semântica.
<abbr title="Significado da sigla"> e </abbr>
Tag para adicionar o significado de uma sigla, onde você pode colocar o significado na parte "title" e sempre que o usuário passar com o mouse por cima aparecerá o significado.
<em> e </em>
Tag utilizada para inserir o itálico em textos, porém, de maneira semântica.
<sup> e </sup>
Insere no texto um expoente, caso você precise digitar fórmulas matemáticas.
<sub> e </sub>
Insere um índice no texto, o número que fica na parte de baixo de uma fórmula química, H2O.
<del> e </del>
Funciona como se estivesse deletando algum trecho, deixa o efeito de riscado no texto.
<ins> e </ins>
Deixa um sublinhado no texto, como se fosse uma lacuna.
<mark> e </mark>
Marca uma parte do texto, como se fosse um marca-texto mesmo.
<small> e </small>
Faz com que a fonte da letra se altere deixando-a menor.
<code> e </code>
Caso você precise escrever algum código de outra linguagem, por exemplo, essa tag vai colocar atributos diferentes ao modo como o conteúdo é exibido.
<pre> e </pre>
Faz com que o conteúdo seja exibido exatamente da mesma forma como está escrita no código.
<hr>
Esta tag faz uma linha na largura total da tela para separar dois trechos que você desejar.
<br>
É uma tag de quebra de linha(break row).
<bdo dir="ltr ou rtl"> e </bdo>
Usado para escrever um trecho de trás para frente(rtl) ou de forma normal(ltr), pelo menos para nós.
<q> e </q>
Usado em caso de necessidade para fazer uma citação no meio de um parágrafo, visualmente essa tag apenas coloca o trecho entre aspas duplas, porém, semanticamente têm sua importância.
<blockquote> e </blockquote>
Funciona como um bloco de citação, ao invés de fazer o papel do anterior que funciona no meio de um parágrafo.
<u> e </u>
Usado caso necessário sublinhar(underline) alguma parte do texto.
<!--Comente qualquer coisa-->
Serve para você adicionar comentários no seu código, tudo o que é inserido entre os hífens vira comentário.

Listas

<ol> e </ol>
Este tipo de lista é a lista ordenada(ordered list), que por padrão tem seus itens enumerados.
<ul> e </ul>
Já esta é a lista desordenada(unordered list), que por padrão não tem seus itens enumerados, apenas uma marcação indicando que aquilo é uma lista.
<li> e <;li>
E este é o item da lista(list item), local onde você colocará o nome dos elementos.
<dl> e </dl>
Agora esta é a lista de definição(definition list), que funciona como um dicionário, este mesmo é um exemplo desta lista.
<dt> e </dt>
A tag acima serve para definir o termo da lista
<dd> e </dd>
Esta tag é usada para fazer a definição do termo.

Links e Âncoras

<a href="URL" target="Indica onde será aberto o link"> e </a>
Esta é a tag responsável por criar uma âncora dentro de um site, que nada mais é do que um link, sendo a parte do href onde o responsável pelo site coloca a URL para onde o usuário deve ser direcionado e a parte entre a abertura e o fechamento das tags onde a ele coloca também o conteúdo que deve ser mostrado a pessoa que estiver consumindo o conteúdo. O target pode ser dentro do próprio site(_self) ou redirecionar para fora(_blank).

Obs: a própria barra de navegação pelo conteúdo do site é formada por várias âncoras.

Áudio, Imagem e Vídeo

<audio src="Caminho para o áudio"> e </audio>
É uma tag usada para inserir áudio no seu site, onde o desenvolvedor colocará o local do arquivo sonoro dentro do "src"
<img src="Caminho para a imagem" alt="Sobre o que se trata a imagem">
Tag usada para colocar imagens em seu site, sendo apenas necessário informar o caminho da imagem no "src" e sobre o que a imagem se trata dentro do "alt" para ajudar o google a entender a imagem além de ajudar deficientes visuais por meio de ferramentas de leitura de site.
<video src="Caminho para o vídeo" poster="Para colocar thumbnail em seu vídeo" controls> e </video>
É uma tag usada para inserir vídeo no seu site, onde o desenvolvedor colocará o local do arquivo de vídeo dentro do "src", é também facultativo o uso do recurso "poster" e controls, que disponibiliza para o usuário os controles do vídeo.

Formulários

<form> e </form>
É a tag onde seu formulário ficará dentro.
<label> e </label>
Esta é a tag responsável pela identificação do input do formulário.
<input type="text" placeholder="Texto que aparece dentro" required>
Esta tag é usada normalmente dentro de um formulário para o usuário colocar o nome dele, sendo basicamente uma caixa de texto onde o cliente pode digitar qualquer coisa. A parte do
<input type="email" placeholder="Texto que aparece dentro" required>
Tag usada para fazer a verificação de email, local onde o usuário pode digitar seu email.
<input type="password" placeholder="Texto que aparece dentro" required>
Esse input serve para o usuário digitar sua senha e automaticamente ocultar a senha.
<input type="radio" value="Resultado recebido ao marcar a caixa" name="Nome da classe" id="Nome do item">
Tipo de caixa de marcar, que só suporta um valor, sendo o "value" o valor recebido pelo banco de dados, o "name" é o nome atribuído a classe e o "id" é a identificação única do item.
<input type="checkbox" value="Resultado recebido ao marcar a caixa" name="Nome do item" id="Nome do item">
Tipo de caixa de marcar, que suporta vários valores, sendo o "value" o valor recebido pelo banco de dados, o "name" é o nome atribuído ao item e o "id" é a identificação única do item.
<select name="Nome do seletor"> e </select>
São as tags de abertura e fechamento de uma lista de itens que o usuário pode selecionar um deles.
<option selected(Para deixar a caixa marcada como padrão)> e </option>
São as tags onde você colocará o conteúdo das opções de escolha entre elas.
<textarea> e </textarea>
É uma área onde o usuário pode digitar uma mensagem maior, como uma mensagem do WhatsApp.
<input type="submit" value="Enviar">
Botão de envio do formulário.
<button> e </button>
Cria um botão de envio de formulário.
